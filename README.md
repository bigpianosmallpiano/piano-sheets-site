# Big Piano Small Piano — SEO Companion Site

A fully automated, statically-generated blog companion to the
[Big Piano Small Piano](https://www.youtube.com/channel/UCsKtOGplqKz3uGt9XD8VexQ)
YouTube channel.

Built with **Astro** + **Tailwind CSS** + **Python** + **GitHub Actions**.

## Quick Start (local development)

### 1. Prerequisites
- Node.js ≥ 20
- Python ≥ 3.10
- A YouTube Data API v3 key

### 2. Install dependencies

```bash
npm install
pip install requests
```

### 3. Fetch your video data

```bash
export YT_API_KEY="your_youtube_api_key_here"
python fetch_and_parse.py
```

This writes `src/data/videos.json` and `src/data/scores.json`.

### 4. Start the dev server

```bash
npm run dev
# → http://localhost:4321
```

### 5. Build for production

```bash
npm run build
# Output in ./dist/
```

## Deployment (Netlify + GitHub Actions)

### One-time setup

1. **Push this repo to GitHub.**

2. **Add GitHub Secrets** (Settings → Secrets and variables → Actions):
   | Secret | Value |
   |--------|-------|
   | `YT_API_KEY` | Your YouTube Data API v3 key |
   | `NETLIFY_AUTH_TOKEN` | Netlify → User settings → Applications → New access token |
   | `NETLIFY_SITE_ID` | Netlify → Site settings → General → Site ID |

3. **Create a Netlify site** and connect it to your GitHub repo, or let the
   Actions workflow deploy directly (the workflow uses the Netlify CLI action).

4. **Update `astro.config.mjs`** — replace `https://bigpianosmallpiano.com`
   with your actual domain.

5. **Add your domain** in Netlify → Domain settings → Add custom domain.
   Enable HTTPS (free via Let's Encrypt).

6. **Submit your sitemap** to Google Search Console:
   `https://yourdomain.com/sitemap-index.xml`

### How automation works

Every day at 06:00 UTC, GitHub Actions:
1. Runs `fetch_and_parse.py` → updates `src/data/videos.json` + `scores.json`
2. Builds the Astro site → generates new HTML for any new videos
3. Commits the updated JSON to Git (tracked history)
4. Deploys the `dist/` folder to Netlify

New video uploaded → within 24 h a new SEO-indexed page appears automatically.

## Customisation

### Adding score link patterns
Edit `SCORE_LINK_PATTERNS` in `fetch_and_parse.py` to add new platforms.

### Adjusting credit patterns
Edit `CREDIT_PATTERNS` in `fetch_and_parse.py`. The regex captures the text
after keywords like "Arranged by:", "Score by:", etc.

### Changing the colour theme
All design tokens are in `tailwind.config.mjs` under `theme.extend.colors`.
The palette uses `ivory` (warm white), `ebony` (near-black), and `gold`.

## SEO Notes

- Every video page has a unique `<title>`, `<meta description>`, Open Graph
  tags, Twitter Card, and a **JSON-LD VideoObject** schema.
- The `/scores` page targets "free piano sheet music" long-tail keywords and
  has a **JSON-LD CollectionPage** schema.
- Astro's `@astrojs/sitemap` integration auto-generates `/sitemap-index.xml`.
- All pages are static HTML — Google can fully crawl and index them.
- YouTube thumbnails are used as `og:image` for rich social preview cards.
